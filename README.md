## Usage

git clone https://github.com/elfinsanjaya12/Server-Tiket-TRC.git :

```bash
$ git https://github.com/elfinsanjaya12/Server-Tiket-TRC.git
```

cd Server-Tiket-TRC

```bash
$ cd Server-Tiket-TRC
```

npm install

```bash
$ npm install
```

npm install -g sequelize

```bash
$ npm install -g sequelize
```

npm db_start

```bash
$ npm db_start
```

npm install -g nodemon

```bash
$ npm install -g nodemon
```

npm dev

```bash
$ npm dev
```

### DOCS API

```bash
localhost:3000/api/v1
```
